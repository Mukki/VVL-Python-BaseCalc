EXÉCUTION DE L'APPLICATION
-----------------------------------------------------------
Pour exécuter l'application, exécuter la commande :

	>>> python BaseCalc.py

L'application doit se trouver dans le répertoire de travail
Python où la commande sera elle-même entrée; si Python est
ajouté aux variables d'environnement, le répertoire n'a pas
d'importance pourvu que l'invite de commande soit au même
répertoire que l'application.


UTILISATION DE LA CALCULATRICE
-----------------------------------------------------------
1 - L'entrée peut se faire au clavier ou avec la souris

2 - Pour afficher le résultat, appuyer sur '='

3 - Pour effacer le calcul en cours, appuyer sur 'c'

4 - Il est possible de  poursuivre un calcul deja en cours
    après l'ajout d'un opérateur, l'opérateur est alors
    ajouté au dernier résultat.

5 - Pour calculer avec un nombre négatif, appuyer sur '-'
    avant le nombre, par exemple 3--4=-1 et -0.5+1=0.5

6 - Affichage de l'historique : Option->Historique

7 - Suppression de l'historique : Option->Effacer l'historique
